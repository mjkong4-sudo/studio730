## How to Set It Up

### Step 1: Click on Prisma Postgres
- Click the card that says "Prisma Postgres"
- It should say "Instant Serverless Postgres"

### Step 2: Configure
- Name your database (e.g., `studio730-db`)
- Choose a region (closest to you)
- Click "Create" or "Continue"

### Step 3: Get Connection String
After creation:
1. Click on your database in Storage
2. Look for "Connect" tab
3. Find `POSTGRES_PRISMA_URL` or connection string
4. Copy it

### Step 4: Add to Environment Variables
1. Go to Settings â†’ Environment Variables
2. Add `DATABASE_URL`
3. Paste the connection string
4. Select all environments
5. Turn Sensitive ON
6. Save
